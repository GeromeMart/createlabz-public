## General information
Liquid crystal displays (LCDs) are commonly used for displaying data in devices such as calculators, microwave, printers and other electronics devices. 

## Hardware needed
* Arduino Uno
* 128 x 64 LCD Module
* Breadboard
* 10k Potentiometer
* Jumper Cables
