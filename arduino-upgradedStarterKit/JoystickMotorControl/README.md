## General information
This project was created as part of the Arduino Upgraded Kit tutorials for createlabz. This aims to provide a brief introduction on the basics of Arduino.

## Hardware needed
* Arduino Uno
* Stepper Motor
* Stepper Motor Driver Board
* 9g Servomotor
* Passive Buzzer
* XY Joystick Module
* Jumper Cables
