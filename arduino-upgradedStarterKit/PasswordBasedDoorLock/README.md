## General information
This project was created as part of the Arduino Upgraded Kit tutorials for createlabz. This aims to provide a brief introduction on the basics of Arduino.

## Hardware needed
* Arduino Uno
* 220 ohm resistor x4
* Breadboard
* Passive Buzzer
* 74HC595
* 4-Digit Display Tube (7-Segment Common Anode)
* 4x4 Keypad
* 9g Servomotor
* Jumper Cables
